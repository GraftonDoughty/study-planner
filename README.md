# my-first-lab
My first web application

## Target Users
Students managing coursework and GPA.

## Features
- Create tasks with title, due date
- View all tasks in a calender
- Mark tasks as complete
- View GPA
- Deletion of Tasks
- Shows active courses

## Tech Stack
- Next.js (React framework)
- Deployed on Vercel

## User Stories
- "As a student, I want to calanderize all my work, so I can track my work and stay on top of my assignments."
- "As a student, I want to mark my completed work, so I can see what I have left to do."
- "As a student, I want to delete my semester, so I can start fresh for my next semester."
- "As a student, I want to see my classes, so I can visualize my work load and see where my classes will be & when they start."
- "As a student, I want to see my GPA, so I can see how well I rank at my school and see what colleges I can go to."
